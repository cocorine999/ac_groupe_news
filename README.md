
# Enfold project

This a blogging application name Enfold

## Instruction

- composer install

- Generate app key
```
php artisan key:generate
```

- set db name

- set db access

- migrate database architecture
```
php artisan migrate
php artisan db:seed
```

- Install nodejs
```
npm install

npm run dev

```

- Run application
```
php artisan serve
```

- Login

- Add categories

- Add Tags

- Add 20 Posts

- Generate storage symbolic link
```
php artisan storage:link
```
